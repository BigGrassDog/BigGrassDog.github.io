连接 github 与本地

  • 在 git bash 中根据注册 github 的信息设置用户名和邮箱
  • 生成密钥 SSH key

打开 github，在头像下面点击 settings， 再点击 SSH and GPG keys，新建一个 SSH。

  • 输出密钥 并复制 添加至 github
  • 验证是否连接成功
